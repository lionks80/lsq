lsqPaging

jQuery Plugin.

Demo

           $('#lsqPaging_demo').lsqPaging({
                "start" : 59,
                "limit" : 15,
                "totalCount" : 300,
                "displayCount" : 9,
                "trigger" : function(page) {
                    alert(page);
            }})

Install

this demo use bootstrap css

        <script src="lsqPaging.js"></script>

options

Name Type Description
start integer 현재 게시물 시작 번호
limit integer 페이지 출력 수
totalCount integer 전체 게시물 수
displayCount integer 페이징 버튼 표시 수
trigger function(page) 페이징 버튼들을 클릭했을 시 동작
