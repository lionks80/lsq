lsq

lsq는 자바스크립트 클래스 오브젝트입니다. lsq는 간단하고 편리한 유틸리티의 집합으로 string, form, ajax, communication manager와 같은 클래스 오브젝트 및 함수로 이루어져 있습니다.

lsqGrid

lsqGrid는 간단한 테이블 그리드 입니다. 사용자가 지정한 css를 기반으로 돌아가기에 어떤 환경에서도 디자인을 적용할 수 있으며 셀렉트, 클릭등의 이벤트를 지원합니다.

lsqPaging

lsqPaging은 페이징 툴바입니다. 사용자가 간단히 기본정보를 입력 후 호출하게 되면 자동으로 페이징 툴바를 만들어줍니다. 이때 페이징 버튼 클릭 시 이벤트는 사용자가 트리거로 정의할 수 있으며 이를 통해 웹 개발 시 개발자의 편의대로 동작하게 됩니다.
