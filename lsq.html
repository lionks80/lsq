lsq.ajax - lsq Ajax request

Methods

Return Method Description
boolean lsq.ajax(method, requestUrl, data, contentType, async, beforeSend, complete) jquery ajax 호출 후 결과를 jsonObject 타입으로 반환

Options

Name Description
method jQuery Argument 요청방식 - 기본값: 'GET'
requestUrl jQuery Argument 요청 URL
contentType jQuery Argument 데이터 전송 타입 - ex) 'application/json'
data jQuery Argument 데이터 - QueryString or POST data
async jQuery Argument ajax 호출 방식 - 기본: 동기식
beforeSend jQuery Argument 호출전 실행 함수
complete jQuery Argument 호출 후 실행 함수

lsq.cm - lsq Communication Manager

Methods

Return Method Description
void lsq.cm.createRestCM(baseUrl, listUrl, getUrl, createUrl, updateUrl, deleteUrl) create Restfull Communication Manager

Example


// create testCM
var testCM = lsq.cm.createRestCM('/book');

//[GET request with queryString data] "/book" with data
testCM.list(data);

//[GET request with queryString data] "/book/123"
testCM.get('123', data);

//[POST request with 'application/json' type data] "/book"
testCM.create(key, data);

//[PUT request with 'application/json' type data] "/book/123"
testCM.update(123, data);

//[DELETE request] "/book/123"
testCM.del('123');
